# OKRs 2026 — Implementação & Consultoria CPJ-3C

Dashboard interativo de acompanhamento semanal de OKRs para apresentação em 1:1.

## Como usar

1. Baixe ou clone este repositório
2. Abra o arquivo `okr_dashboard_barbara.html` diretamente no navegador (Chrome, Edge ou Firefox)
3. Nenhuma instalação ou servidor necessário — funciona 100% offline

## Funcionalidades

- **3 Objetivos Estratégicos** com seus Key Results reais
- **Atualização de valores** por KR com barra de progresso automática
- **Status por KR**: Não iniciado / Em andamento / Concluído / Atrasado
- **Histórico semanal por KR**: registre o que fez e os próximos passos a cada semana
- **Gráficos de trajetória**: alvo vs. realizado para implementação e consultoria
- **Impressão / PDF**: botão direto para gerar PDF da visão atual

## Estrutura dos OKRs

| Objetivo | Foco | KRs |
|---|---|---|
| Obj 1 | Prazo de implementação | 4 KRs |
| Obj 2 | Qualidade no go live | 5 KRs |
| Obj 3 | Consultoria (horas e satisfação) | 5 KRs |

## Rotina sugerida

Toda semana antes da 1:1 com o diretor:
1. Abra o arquivo no navegador
2. Atualize o valor atual de cada KR que avançou
3. Expanda o histórico semanal e adicione o registro da semana
4. Apresente na reunião ou exporte como PDF

## Tecnologias

- HTML5 + CSS3 + JavaScript puro
- [Chart.js 4.5](https://www.chartjs.org/) via CDN (apenas para os gráficos)
- Sem frameworks, sem dependências locais
